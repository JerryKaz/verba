# Verba — Professional Linguistic Environment

Verba is an AI-powered linguistic workstation engineered for professional translation, semantic analysis, and multilingual communication. Built with a modern full-stack architecture and powered by Google Gemini, Verba transforms documents, conversations, and visual text into structured, context-aware insights.

---

## Overview

Verba combines real-time translation, intelligent document analysis, OCR-powered text extraction, and conversational AI into a unified productivity platform. The application is optimized for performance, accessibility, and cross-device usability while maintaining a professional-grade user experience.

---

## Features

### AI Linguistic Assistant
A conversational AI assistant powered by Google Gemini capable of:

- Advanced translation refinement
- Grammar and syntax analysis
- Idiom interpretation
- Tone and contextual language suggestions
- Semantic explanation for complex phrases

---

### Semantic Document Analysis
Professional document intelligence for:

- PDF processing
- DOCX extraction
- Image-based text analysis

Capabilities include:

- Executive summaries
- Key theme extraction
- Semantic complexity estimation
- Contextual content interpretation
- Structured document insights

---

### Real-Time Translation Engine
A multilingual translation system supporting dozens of languages with:

- Instant translation
- Phonetic pronunciation guides
- Speech synthesis integration
- Context-aware linguistic refinement
- High-speed response architecture

---

### OCR Document Scanner
Integrated OCR functionality enables users to:

- Extract text from uploaded images
- Scan live camera content
- Translate visual text in real time
- Digitize printed documentation

---

### Glossary & Terminology Overrides
Custom linguistic controls allow users to define persistent terminology mappings for:

- Brand terminology
- Technical vocabulary
- Industry-specific language
- Preferred translation outputs

---

### Hybrid Offline Support
Verba includes intelligent local caching mechanisms to preserve core functionality in unstable or low-connectivity environments.

# Installation

Clone the repository:

```bash
git clone https://github.com/JerryKaz/verba.git
```

Navigate into the project:

```bash
cd verba
```

Install dependencies:

```bash
npm install
```

---

# Environment Variables

Create a `.env` file in the root directory:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

---

# Running the Project

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Deploy to GitHub Pages:

```bash
npm run deploy
```
